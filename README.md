# 小水滴 · GitHub Pages 静态版

这是可以直接上传到 GitHub Pages 的版本。

## 文件
- `index.html`
- `style.css`
- `app.js`

## 怎么上线到 GitHub Pages
1. 新建一个 GitHub 仓库，比如 `xiaoshuidi`
2. 把这三个文件上传到仓库根目录
3. 打开仓库 `Settings`
4. 左边点 `Pages`
5. `Source` 选择 `Deploy from a branch`
6. Branch 选择 `main`，文件夹选择 `/root`
7. 保存
8. 等一两分钟，GitHub 会给你一个网址

## 说明
- 这个版本不需要付费网站。
- 登录、待办、打卡、日历等数据保存在浏览器 localStorage。
- GitHub Pages 是静态网站，所以 Google/微信/QQ 真登录、多人实时自习室、充值解锁目前是预留入口。
- 后面如果要真正多人同步，可以接 Supabase 免费版。
